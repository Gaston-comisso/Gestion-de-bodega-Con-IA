Registrar Datos de Uva

La Ciencia detrás del Vino

Bienvenido al sistema de gestión de uvas, donde los enólogos pueden registrar y analizar datos importantes para la elaboración del vino. Nuestro objetivo es proporcionar una herramienta eficaz y confiable para capturar la información relevante sobre la uva.

Datos Clave que se Pueden Registrar

Fecha de Cosecha

Registra la fecha de cosecha de las uvas para realizar un seguimiento preciso del proceso de producción.

Tipo de Uva

Selecciona el tipo de uva utilizado, para mantener un control detallado de la variedad empleada en la producción de vino.

Cantidad

Ingresa la cantidad de uvas cosechadas para gestionar mejor los recursos y la producción.

Litros de Vino

Indica los litros de vino producidos para un control detallado del rendimiento del proceso de vinificación.
